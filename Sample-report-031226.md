醫療器材審查指引：骨外固定器 (Orthopedic External Fixation Devices)
1. 適用範圍 (Scope)
本指引適用於衛生福利部食品藥物管理署 (TFDA) 查驗登記申請案中，所有「骨外固定器 (Orthopedic External Fixation Devices)」相關產品。此品項涵蓋以下主要構成元件：

錨定元件 (Anchorage elements)：用於固定於骨骼上的組件。
橋接元件 (Bridge elements)：用於連接錨定元件並提供結構支撐的組件。
接合器 (Connectors)：用於連接橋接元件與其他組件的組件。
本品項對應之醫療器材分類分級品項為：

N.3030：骨外固定系統 (External Fixation System, Bone)
N.3040：骨外固定器組件 (External Fixation Component, Bone)
本指引旨在確保該等醫療器材之安全與效能符合現行法規要求。

2. 審查重點 (Review Focus)
審查員將依據以下清單，全面評估申請案之技術文件與測試報告：

一、 產品敘述與規格 (Product Description & Specs)

用途說明：是否具備完整且明確的臨床適應症與預定用途說明？
組件目錄：是否表列所有系統組件清單，包含其料號、名稱與數量？
幾何尺寸：是否提供詳細的產品幾何尺寸與規格描述？
工程圖：是否檢附完整的各主組件尺寸工程圖，包含關鍵尺寸與公差？
組合方式：是否詳細描述各組件之設計特徵，以及主組件間的組合與鎖固方式？
材質證明：是否載明所有與人體接觸或具結構功能之材質，並標示所符合之國際材質標準 (如 ASTM/ISO) 及提供符合性證明？
等同性比較：是否與已上市之類似品 (Predicate device) 進行預定用途、設計、材質、性質等列表比較？
差異性評估：針對與類似品之相異處，是否提出合理的工程技術及臨床結果基礎說明，證明差異性不會產生新風險或對安全效能有負面影響？
二、 生物相容性評估 (Biocompatibility Evaluation) (註：如為常規植入金屬材質並附證明，且已有完整上市證明與文獻支持其生物相容性，則可豁免部分或全部試驗；否則，需依據 ISO 10993 系列標準執行以下必要試驗，並提交完整報告)：

ISO 10993-5：細胞毒性試驗 (Cytotoxicity)
ISO 10993-10：過敏試驗 (Sensitization)
ISO 10993-10：刺激或皮內刺激試驗 (Irritation/Intracutaneous)
ISO 10993-11：急性系統毒性 (Acute systemic toxicity)
ISO 10993-11：亞慢性毒性 (Subchronic toxicity)
ISO 10993-3：基因毒性 (Genotoxicity)
ISO 10993-6：植入試驗 (Implantation)
ISO 10993-11：慢性毒性 (Chronic toxicity) (若為長期植入)
ISO 10993-3：致癌性 (Carcinogenicity) (若為長期植入且有疑慮)
三、 滅菌確效 (Sterilization Validation)

SAL 證明：無菌包裝供應者，是否證明其無菌保證水準 (SAL) 符合國家或國際標準要求 (例如 SAL ≤ 10⁻⁶)？
滅菌標準：是否依據產品之滅菌方式採用對應的國際標準執行確效，並提交完整確效報告？
濕熱滅菌：ISO 17665-1
環氧乙烷 (EO) 氣體滅菌：ISO 11135
輻射滅菌：ISO 11137
四、 機械性質評估 (Mechanical Properties) (需依據 ASTM F1541 標準及其他相關標準執行，並提交完整試驗報告)：

剛性測試：剛性或屈折測量 (Rigidity or deflection measurement) 試驗結果。
靜態破壞：裝置的靜態破壞試驗 (Static failure of the device) 試驗結果。
疲勞與鬆脫：各組件間疲勞破壞及鬆脫試驗 (Fatigue failure and loosening between parts) 試驗結果。
五、 特定風險分析與動物試驗 (Special Analysis/Animal Testing) 若產品符合以下任一條件，必須進行額外評估或試驗，並提交相關證明文件或報告：

動態機能：產品標示為脊椎用，或於骨頭修復期間具備動態機能者，需評估其動態負荷下的性能與安全性。
結構硬度：與類似產品相較，具有顯著較低的結構性硬度者，需提供額外數據或分析證明其在預期用途下的足夠強度與穩定性。
MRI 相容性：產品宣稱與核磁共振造影 (MRI) 環境相容者，需依據 ASTM F2052 (MR Safe/Conditional/Unsafe 標示) 等相關標準進行評估與測試，並提供報告。
3. WOW 3 模擬退件風險評估：常見缺失與風險預測
以下列出骨外固定器查驗登記申請案中，常見導致退件或要求補件的缺失，供製造商參考：

文件完整性不足：
風險預測：高。
常見缺失：
未附出產國許可製售證明 (CFS)，或所附證明文件不符合規定 (如過期、未載明申請產品名稱、未經我國駐外單位驗證)。
申請資料內容不完整，例如未提供所有組件清單、缺乏完整的工程圖或關鍵尺寸。
審查費用支付憑證不明確或遺漏。
產品敘述與規格不明確：
風險預測：中高。
常見缺失：
用途說明過於籠統，未明確指出所有臨床適應症及預期用途。
未詳細描述各組件之設計特徵及主組件間的組合與鎖固方式，導致審查員無法判斷產品設計的合理性與安全性。
材質證明不完整，或未標示所符合的國際材質標準 (如僅提供供應商COA，但未說明符合ASTM/ISO何種規範)。
等同性與差異性評估不足：
風險預測：高。
常見缺失：
與類似品之比較表未涵蓋所有關鍵項目 (如預定用途、設計、材質、性質、滅菌方式等)。
未能針對與類似品之相異處 (例如新設計、新材質) 提出合理的工程技術或臨床結果基礎說明，以證明不會產生新風險或對安全效能有負面影響。僅宣稱「差異不影響安全性」但無佐證。
生物相容性試驗不符合要求：
風險預測：高。
常見缺失：
未執行所有必要的 ISO 10993 系列生物相容性試驗，或試驗報告不符合標準要求 (例如試驗方法不正確、結論不明確)。
若宣稱豁免生物相容性試驗，但未能提供充足的常規植入金屬材質證明，或其豁免理由不充分 (例如引用之文獻與產品材質/接觸時間不完全吻合)。
滅菌確效資料不完整或不符合標準：
風險預測：中高。
常見缺失：
未提供 SAL ≤ 10⁻⁶ 的證明文件，或報告中SAL數據未達標。
滅菌確效報告不完整，或未依據所宣稱的滅菌方式採用對應的 ISO 標準執行確效 (例如，EO滅菌未提供殘留量測試)。
機械性質評估不符要求：
風險預測：高。
常見缺失：
未依據 ASTM F1541 執行所有要求的機械性質試驗，或報告數據不完整 (例如缺少疲勞測試)。
試驗方法或條件與產品實際應用情況不符，未能充分模擬臨床使用情境。
特定風險分析與動物試驗缺乏：
風險預測：高。
常見缺失：
若產品符合特定條件 (如脊椎用動態機能、顯著較低結構硬度、MRI 相容性宣稱)，但未進行額外評估或動物試驗。
MRI 相容性宣稱但未提供充分的試驗數據證明 (例如缺乏磁場誘導力、電磁場感應發熱或射頻場加熱測試)。

歐盟 MDR 分類：根據 MDR Annex VIII, Rule 8，侵入性器械及長期植入性器械通常歸類為 Class IIb。骨外固定器透過骨釘侵入人體並與骨骼長期接觸 (通常超過30天用於骨骼修復)，故一般歸類為 Class IIb。若其預定用於脊椎或對中樞神經系統有接觸，或含有藥物、動物組織等特殊情況，則可能升級為 Class III。

流程說明：

申請人提交申請：製造商或代理商提交查驗登記申請文件。
文件完整性初審：TFDA 進行初步資料審查，確認文件齊備與格式符合要求。若有缺失則通知申請人補件。
醫療器材分類分級：根據產品風險程度，將醫療器材分為 Class I (低風險)、Class II (中風險) 或 Class III (高風險)。骨外固定器因其侵入性與長期接觸，通常歸類為 Class II 或 Class III。
技術性審查：
Class I：進行簡化技術審查。
Class II：進行標準技術審查，涵蓋本指引的「審查重點」各項要求。
Class III：進行嚴謹技術審查，除了本指引要求外，可能還需提供臨床試驗資料，並可能召開專家諮詢會議。
骨外固定器審查路徑：針對骨外固定器 (N.3030/N.3040)，審查重點會聚焦於「產品敘述與規格」、「生物相容性評估」、「滅菌確效」、「機械性質評估」及「特定風險分析與動物試驗」。對於 Class III 產品，可能進一步要求臨床資料審查。
符合性判定：審查員根據所有提交資料與審查結果，判斷產品是否符合安全與效能要求。若不符合，則通知申請人補件或退件。
核發許可證：若產品經審查確認符合所有法規要求，TFDA 將核發醫療器材許可證。
